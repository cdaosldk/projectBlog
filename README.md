# projectBlog : 기본 게시판 CRUD에 다양한 기능 구현 추가 프로젝트

# 개발환경

- Spring Boot Version : 2.7.6

- Java 17

- Gradle

- Redis 2.7.6

- AWS S3

# 개발기간

2022.12, 이후 꾸준하게 연습하며 업데이트 중

# 현재까지 구현한 기능

- 게시판 CRUD
- 댓글 CRUD
- AWS S3
- Spring Security
- CI/CD
- Redis를 활용한 RefreshToken
- 도커화
